# Database-Design
This is a course project for Database course which requires students to combine a website and MySQL database and explain whole procedure

# Introduction

# 💇‍♂️ 美发沙龙管理系统 (Beauty Salon System)
基于 PHP + MySQL 的数据库课程设计。

## 🌟 项目亮点
- **三角色管理**：集成管理员(Admin)、发型师(Staff)与客户(Customer)三端逻辑。
- **权限校验**：完善的 Session 守护机制（`session_guard.php`）。
- **数据分析**：管理员可查看发型师业绩及 VIP 统计。

## 📁 目录指南
- `/admin`: 管理员端后台及界面
- `/hairdresser`: 发型师预约及排班管理
- `/auth`: 统一权限验证与登录模块
- `/sql`: 数据库 E-R 图及 SQL 建表脚本
